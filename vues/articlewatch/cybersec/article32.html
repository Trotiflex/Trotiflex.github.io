Zoom sur les dix vulnérabilités critiques ciblant les LLM
1 septembre 2023

L'Open Worldwide Application Security Project (OWASP) a publié une liste des dix vulnérabilités les plus critiques affectant les grands modèles de langage (LLM). Ces vulnérabilités incluent :

  • Les injections rapides : attaques permettant d'injecter du code malveillant dans les modèles.
  • Les fuites de données : risques liés à la divulgation non autorisée d'informations sensibles.
  • Le sandboxing inadéquat : défaut d'isolation des modèles, exposant le système à des attaques.
  • L'exécution de code non autorisée : possibilité pour des attaquants d'exécuter du code arbitraire sur le système cible.

Ces vulnérabilités peuvent avoir des impacts significatifs sur la sécurité et l'intégrité des systèmes utilisant des LLM. Il est essentiel pour les organisations de les identifier et de mettre en œuvre des mesures de sécurité appropriées pour les atténuer.

Source : Le Monde Informatique
