Une version Linux du ransomware Cicada cible les serveurs VMware ESXi
1ᵉʳ septembre 2024

Une nouvelle opération de ransomware en tant que service (RaaS) se faisant passer pour l'organisation légitime Cicada 3301 a été identifiée. Cette opération a déjà répertorié 19 victimes sur son portail d'extorsion, attaquant rapidement des entreprises à l'échelle mondiale. Le ransomware cible spécifiquement les serveurs VMware ESXi en utilisant une version Linux de son malware. Les chercheurs ont observé que cette version est conçue pour chiffrer les environnements VMware ESXi lors d'attaques ciblées contre des entreprises. Les administrateurs de ces systèmes sont donc conseillés de renforcer leur sécurité pour se protéger contre cette menace croissante.

Source : BleepingComputer
